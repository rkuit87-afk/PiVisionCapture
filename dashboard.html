🔄 Workflow

  1. PLC Trigger → Sends HTTP POST to Pi:8888
  2. Main App → Captures frame from camera stream
  3. Storage → Saves JPG + JSON metadata
  4. Result → Board image stored in timestamped folder
All components run headless on the Pi. Control via REST API from engineering PC.
